## LabEcommerce

### Descrição
- API para um ecommerce
- Principais funcionalidades: cadastrar/visualizar usuários, produtos e histórico de compras.

### Tecnologias utulizadas:
- Typescript
- Node.js
- Dotenv
- Express
- MySQL
- Knex
